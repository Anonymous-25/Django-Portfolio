👋 Welcome to Django Portfolio CMS

Your database is empty. Follow this step-by-step guide to set up your portfolio 🚀

📦 Step 1: Download the Project

Open your terminal or command prompt and run:

git clone https://github.com/Anonymous-25/Django-Portfolio.git

This command downloads the project to your computer.

cd Django-Portfolio

This moves you into the project folder.

🐍 Step 2: Create Virtual Environment

Create a virtual environment (isolated Python environment):

python -m venv venv

Activate it:

# Windows venv\Scripts\activate # Mac/Linux source venv/bin/activate

📥 Step 3: Install Dependencies

Install required Python packages:

pip install -r requirements.txt

This installs Django and other libraries used in the project.

🗄 Step 4: Setup Database

Run migrations to create database tables:

python manage.py migrate

This creates all necessary tables like Profile, Skills, Projects, etc.

👤 Step 5: Create Admin User

Create a superuser account:

python manage.py createsuperuser

Enter username, email, and password when asked.

🚀 Step 6: Run Server

python manage.py runserver

Open your browser and go to:

http://127.0.0.1:8000/

🔐 Step 7: Login to Admin Panel

Click below and login using the superuser you created:

Go to Admin Panel

🧩 Step 8: Add Your Data

Start adding data in this order:

  • Profile (most important)
  • Skills
  • Experience
  • Education
  • Projects
  • Articles

Once you add any data, your main portfolio UI will automatically appear.

⚡ How It Works

  • No data → You see this guide page
  • Any data added → Main portfolio UI loads
